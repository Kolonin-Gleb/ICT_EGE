Библиотека ipaddress для 13 задачи на КЕГЭ.

— Функция ip_network()
Определяет сеть по IP-адресу и маске. В нестрогом режиме (второй параметр 0) может принимать в качестве IP-адреса любой IP-адрес из описываемой сети.

При выводе результата будет выведен адрес сети и количество единиц в маске через слеш.

С помощью данной функции можно найти все адреса сетей для различных масок, например с помощью цикла на рис.2

— net.netmask и net.hosts()
Возвращают соответственно маску сети и IP-адреса всех узлов сети. В список узлов, что логично, не входят адрес сети и широковещательный адрес.

— Сравнение сети и проверка вхождения IP-адреса в список узлов
Операторы, которые позволяют удобно проверить ограничения из задач.

net1 == net2 вернет истину, если в net1 и net2 описана одна и та же сеть.
ip in net.hosts() вернет истину, если IP-адрес входит в список узлов сети.

Библиотека имеет и другие интересные методы. В этом посте рассмотрены наиболее актуальные для ЕГЭ.
